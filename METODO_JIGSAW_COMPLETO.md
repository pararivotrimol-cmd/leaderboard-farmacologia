# Método Jigsaw Completo para Farmacologia I - UNIRIO

## 📚 Visão Geral

O **método Jigsaw** é uma estratégia de aprendizagem cooperativa que promove responsabilidade individual e coletiva. Nesta implementação para Farmacologia I, os alunos serão divididos em **Grupos de Especialistas** (estudam um tópico em profundidade) e depois em **Grupos Jigsaw** (ensinam uns aos outros sobre todos os tópicos).

---

## 🎯 Objetivos de Aprendizagem

### Competências Desenvolvidas
- **Conhecimento Profundo:** Domínio completo de um tópico farmacológico específico
- **Ensino Colaborativo:** Capacidade de explicar conceitos complexos de forma clara
- **Aprendizagem Integrada:** Compreensão holística de 6 tópicos farmacológicos
- **Responsabilidade Compartilhada:** Cada aluno é responsável por ensinar e aprender
- **Comunicação Científica:** Apresentação clara de conceitos farmacológicos

---

## 📊 Estrutura Geral do Jigsaw

### **Fase 1: Grupos de Especialistas (Semanas 5-6)**
- Alunos são divididos em 6 grupos de especialistas
- Cada grupo estuda UM tópico em profundidade
- Duração: 2 semanas
- Atividades: leitura de artigos, discussão, preparação de apresentação

### **Fase 2: Grupos Jigsaw (Semanas 7, 11, 14, 16)**
- Alunos de diferentes especialidades se reúnem
- Cada aluno ensina seu tópico aos colegas
- Duração: 4 encontros ao longo do semestre
- Atividades: apresentações, discussão, avaliação

### **Fase 3: Avaliação Integrada (Semanas 8, 17)**
- Provas individuais (P1, P2) incluem todos os 6 tópicos
- Avaliação de desempenho individual e em grupo
- Prêmios e reconhecimento

## 🔬 Os 6 Tópicos Jigsaw e Artigos

### **1. Interação Medicamentosa (Drug-Drug Interactions)**

**Artigo Recomendado:**
- Título: "Navigating Drug–Drug Interactions in Clinical Drug Development: A Tutorial"
- Autores: Lohy Das, J., et al.
- Publicação: ASCPT (2025)
- Link: https://ascpt.onlinelibrary.wiley.com/doi/full/10.1111/cts.70342

**Conceitos-chave:**
- Mecanismos de interação (farmacocinética vs farmacodinâmica)
- Inibição/indução de CYP450
- Exemplos clínicos: Warfarina + AINEs, Estatinas + Macrolídeos
- Ferramentas práticas (Micromedex, DrugBank)

**Duração de Estudo:** 5 horas
**Materiais:** Artigo (10 páginas), slides resumo, casos clínicos

---

### **2. Alergia Medicamentosa (Drug Hypersensitivity)**

**Artigo Recomendado:**
- Título: "Accurately describing drug allergies and adverse drug reactions"
- Publicação: PMC/NIH (2026)
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC12743412/

**Conceitos-chave:**
- Classificação de Gell e Coombs (Tipo I-IV)
- Anafilaxia induzida por drogas
- Drogas mais alergênicas (penicilinas, sulfonamidas, AINEs)
- Diagnóstico e manejo de emergência

**Duração de Estudo:** 5 horas
**Materiais:** Artigo (12 páginas), fluxograma de diagnóstico, protocolo de manejo

---

### **3. Farmacodependência de Drogas de Abuso**

**Artigo Recomendado:**
- Título: "Substance use disorder: above and beyond addiction"
- Publicação: Frontiers in Pharmacology (2024)
- Link: https://www.frontiersin.org/journals/pharmacology/articles/10.3389/fphar.2024.1426351/full

**Conceitos-chave:**
- Mecanismos neurobiológicos da dependência (dopamina, recompensa)
- Drogas de abuso: opioides, estimulantes, álcool
- Tratamentos farmacológicos: metadona, buprenorfina, naltrexona
- Síndrome de abstinência

**Duração de Estudo:** 5 horas
**Materiais:** Artigo (15 páginas), diagrama de receptores, protocolos de tratamento

---

### **4. Canabinoides (Cannabis Pharmacology)**

**Artigo Recomendado:**
- Título: "Therapeutic Potential for Cannabis and Cannabinoids 2.0"
- Autores: Raup-Konsavage, W.M., et al.
- Publicação: PMC/NIH (2025)
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC12730434/

**Conceitos-chave:**
- Farmacologia de THC vs CBD
- Receptores canabinoides (CB1, CB2)
- Aplicações terapêuticas: dor crônica, epilepsia, esclerose múltipla
- Efeitos adversos e considerações clínicas

**Duração de Estudo:** 5 horas
**Materiais:** Artigo (14 páginas), estruturas químicas, aplicações clínicas

---

### **5. Farmacologia dos Emagrecedores (Anti-Obesity Pharmacology)**

**Artigo Recomendado:**
- Título: "Glucagon-like peptide-1 receptor agonists for obesity"
- Autores: Chao, A.M., et al.
- Publicação: PMC/NIH (2026)
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC12803457/

**Conceitos-chave:**
- Mecanismo de ação de GLP-1 agonistas
- Semaglutida vs tirzepatida
- Perda de peso e benefícios metabólicos
- Efeitos colaterais e contraindicações

**Duração de Estudo:** 5 horas
**Materiais:** Artigo (11 páginas), comparação de drogas, dados de eficácia

### **6. Farmacologia dos Quimioterápicos para Câncer**

**Artigo Recomendado:**
- Título: "Recent Advances in Anti-Cancer Drugs"
- Autores: Neophytou, C.M., et al.
- Publicação: PMC/NIH (2025)
- Link: https://pmc.ncbi.nlm.nih.gov/articles/PMC12428732/

**Conceitos-chave:**

## 📊 Sistema de Avaliação

### **Avaliação Individual**

#### **Apresentação no Jigsaw (0-5 PF)**
- Clareza e organização (0-2 PF)
- Domínio do conteúdo (0-2 PF)
- Resposta a perguntas (0-1 PF)

#### **Participação no Jigsaw (0-2 PF)**
- Atenção e respeito (0-1 PF)
- Perguntas e comentários (0-1 PF)

#### **Provas (P1 e P2) (0-20 PF cada)**
- Questões objetivas sobre todos os 6 tópicos
- Casos clínicos integrados
- Questões de aplicação clínica

#### **Atividades Complementares**
- Quiz de revisão (0-3 PF)
- Discussão de casos (0-3 PF)
- Escape Room (0-3 PF)

### **Avaliação em Grupo**

#### **Responsabilidade Compartilhada**
- Todos os membros do grupo de especialistas recebem a mesma nota de apresentação
- Todos os membros do grupo Jigsaw recebem a mesma nota de participação

#### **Avaliação de Pares**
- Cada aluno avalia a qualidade da apresentação dos colegas (0-5 pontos)
- Média das avaliações contribui para nota final (0-2 PF bônus)

#### **Prêmios de Equipe**
- Equipe com melhor desempenho geral (5 PF bônus)
- Equipe com melhor apresentação Jigsaw (3 PF bônus)
- Equipe com melhor participação (2 PF bônus)

### **Fórmula de Cálculo de PF Total**

```
PF Total = Σ(Apresentações Jigsaw) + Σ(Participação Jigsaw) + 
           P1 + P2 + Atividades Complementares + Bônus de Equipe

Máximo: 7 + 7 + 7 + 7 + 7 + 7 + 20 + 20 + 9 + 5 = 91 PF
```

**Normalização para 45 PF (MAX_PF_SEMESTER):**
- PF Total × (45 / 91) = PF Normalizado

```typescript
// Tabela: jigsawTopics
interface JigsawTopic {
  id: number;
  name: string;           // "Interação Medicamentosa"
  description: string;
  articleUrl: string;
  articleTitle: string;
  articleAuthors: string;
  articleYear: number;
  keyPoints: string[];
  studyDuration: number;  // em horas
  createdAt: Date;
}

// Tabela: jigsawExpertGroups
interface JigsawExpertGroup {
  id: number;
  classId: number;        // turma
  topicId: number;        // tópico especialista
  name: string;           // "Especialistas em Interação Medicamentosa - Turma 1"
  description: string;
  maxMembers: number;
  status: "forming" | "active" | "presenting" | "completed";
  presentationDate: Date;
  createdAt: Date;
  updatedAt: Date;
}

// Tabela: jigsawExpertMembers
interface JigsawExpertMember {
  id: number;
  expertGroupId: number;
  memberId: number;       // aluno
  role: "member" | "coordinator" | "presenter";
  presentationScore: number;  // 0-5
  participationScore: number; // 0-2
  createdAt: Date;
}

// Tabela: jigsawHomeGroups
interface JigsawHomeGroup {
  id: number;
  classId: number;
  name: string;           // "Jigsaw Group A - Semana 7"
  description: string;
  status: "forming" | "active" | "completed";
  meetingDate: Date;
  createdAt: Date;
  updatedAt: Date;
}

// Tabela: jigsawHomeMembers
interface JigsawHomeMember {
  id: number;
  homeGroupId: number;
  memberId: number;       // aluno
  topicId: number;        // tópico que este aluno ensina
  presentationScore: number;  // 0-5
  participationScore: number; // 0-2
  peerRating: number;     // 0-5 (avaliação de pares)
  createdAt: Date;
}

// Tabela: jigsawScores
interface JigsawScore {
  id: number;
  memberId: number;
  expertGroupId: number;  // null se não apresentou
  homeGroupIds: number[]; // array de IDs dos grupos Jigsaw
  totalPresentationScore: number;
  totalParticipationScore: number;
  totalPeerRating: number;
  totalJigsawPF: number;
  createdAt: Date;
  updatedAt: Date;
}
```
